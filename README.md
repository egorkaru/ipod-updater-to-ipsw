# ipod-updater-to-ipsw

Simple command-line tool to extract ipsw files from iPod Updater.app.

## Installation

```sh
npm install -g ipod-updater-to-ipsw
```

## How to use

```sh
ipod-updater-to-ipsw --input "<path to Ipod Updater.app>" --output "./ipsw"
```
